# Traderfolio

This is small front-end oriented project working with React, AJAX calls, ES6, input autocomplete and more. Doing something nice and simple like storing and showing your savings in crypto, uses [Cryptocompare](https://www.cryptocompare.com/) API and data for displaying funds and giving access to the user to more than 3500 cryptos, balance for non-existing coins (delivered from API) is not allowed. Balance for every coin is shown individually and a total balance of all coins is shown in USD and BTC.

# Deployed instance

https://traderfolio.herokuapp.com/
